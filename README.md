# testcollab-cli (Deprecated)

> **This package has moved to [`@testcollab/cli`](https://www.npmjs.com/package/@testcollab/cli).**

## Migrate

Uninstall the old package and install the new one:

```bash
npm uninstall -g testcollab-cli
npm install -g @testcollab/cli
```

The `tc` command works exactly the same — no changes to your scripts or CI pipelines beyond updating the package name.
